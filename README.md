# LearnPath AI 🎓

LearnPath AI is a personalized learning platform built with Flask.

## Features
- Student login & registration
- Personalized learning recommendations
- Automated timetable generation
- Progress tracking
- Admin dashboard

## Tech Stack
- Python (Flask)
- SQLite
- HTML, CSS
- Jinja2

## Run locally

pip install flask  
python app.py

Open:
http://127.0.0.1:5000
